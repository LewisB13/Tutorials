# Java Beginner Tasks ☕

## Task 1 — Print Your Name
Create a Java program that prints your full name to the screen using:

```java
System.out.println();
```

---

## Task 2 — Favourite Things
Write a Java program that prints:
- Your favourite game
- Your favourite movie
- Your favourite food

---

## Task 3 — Multiple Print Statements
Create a program that prints 3 different lines of text using 3 separate print statements.

Example:
```java
System.out.println("Hello");
System.out.println("Welcome");
System.out.println("To Java");
```

---

## Task 4 — Break The Code 😈
Remove a semicolon from a working program and run it.

- What error appears?
- Can you fix it?

---

## Task 5 — Edit The Output
Take an existing Java program and change the text being displayed.

Example:
```java
System.out.println("Java is fun!");
```

Change it to something else.

---

## Task 6 — About Me Program
Create a Java class called:

```java
AboutMe
```

The program must print:
- Your name
- Your age
- Your favourite hobby

---

## Task 7 — Welcome Message
Create a Java program that displays:

```text
Welcome to Computing
Java is fun
```

on separate lines.

---

## Task 8 — Try Different Software
Open your Java program in:
- jGRASP
- VS Codium (if installed)

Which one do you prefer and why?

---

## Task 9 — Bracket Chaos 💀
Break the brackets in a Java program on purpose.

Example:
```java
{
}
```

- What happens?
- Can you fix the errors?

---

## Task 10 — Funny Console Program 😂
Create a funny Java program using at least 5 print statements.

Example:
```java
System.out.println("Send help");
System.out.println("I forgot a semicolon");
System.out.println("Java is judging me");
```

Bonus points for making the class laugh 😄
