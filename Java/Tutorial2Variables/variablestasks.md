# Java Variables Tasks ☕

## Task 1 — Create a Name Variable
Create a variable called `name` and store your name inside it.

Example:

```java
String name = "Lewis";
```

---

## Task 2 — Create an Age Variable
Create an integer variable called `age` and store your age.

Example:

```java
int age = 21;
```

---

## Task 3 — Create a Height Variable
Create a double variable called `height` and store your height.

Example:

```java
double height = 1.80;
```

---

## Task 4 — Print a Variable
Create a variable and print it using:

```java
System.out.println();
```

Example:

```java
System.out.println(name);
```

---

## Task 5 — Create a Favourite Game Variable
Create a variable called `game`.

Store your favourite game and print it to the console.

Example:

```java
String game = "Minecraft";
```

---

## Task 6 — Create Multiple Variables
Create variables for:

- Name
- Age
- Favourite Food

Print them all to the screen.

---

## Task 7 — Change a Variable Value
Create a variable and then change its value later in the program.

Example:

```java
int score = 10;

score = 20;
```

Print the updated value.
