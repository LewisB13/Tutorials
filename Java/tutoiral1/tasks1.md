# Java Beginner Exercises: `System.out.print()` and `System.out.println()`

## Task 1: First Output

Write a program that prints:

```text
Welcome to Java Programming!
```

**Learning Goal:** Using `System.out.println()`.

---

## Task 2: Personal Information

Print the following information on separate lines:

```text
Name: John
Age: 18
Country: Ireland
```

**Learning Goal:** Multiple output statements.

---

## Task 3: Using `print()` and `println()`

Produce the output below using both `print()` and `println()`:

```text
Java is fun!
```

**Learning Goal:** Understanding the difference between `print()` and `println()`.

---

## Task 4: Daily Schedule

Print a simple schedule:

```text
Monday
- Math
- English
- Science
```

**Learning Goal:** Formatting text output.

---

## Task 5: Shopping Receipt

Create the following receipt:

```text
Receipt
-------
Bread   €2
Milk    €1
Apple   €3
```

**Learning Goal:** Aligning text in console output.

---

## Task 6: Draw a Square

Print the following shape:

```text
*****
*   *
*****
```

**Learning Goal:** Creating ASCII art using output statements.

---

## Task 7: Personal Introduction

Print a short introduction about yourself using 4–5 lines.

Example:

```text
Hello!
My name is Emma.
I am 17 years old.
I enjoy coding and football.
```

**Learning Goal:** Combining multiple statements into a simple program.

---

## Task 8: Report Header

Display the following header:

```text
====================
   STUDENT REPORT
====================
```

**Learning Goal:** Decorative formatting.

---

## Task 9: Favourite Movies

Print a numbered list:

```text
My Favourite Movies
1. Toy Story
2. Shrek
3. Finding Nemo
```

**Learning Goal:** Creating lists in console output.

---

## Task 10: Game Menu

Create a simple game menu:

```text
=== GAME MENU ===
1. Play
2. Settings
3. Exit
Choose an option:
```

**Learning Goal:** Designing a console interface.
